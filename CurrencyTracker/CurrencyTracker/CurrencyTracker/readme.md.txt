# 📊 CurrencyTracker – Döviz Takip Konsol Uygulaması

CurrencyTracker, Türk Lirası (TRY) bazlı döviz kurlarını takip etmek amacıyla geliştirilmiş basit bir C# konsol uygulamasıdır.  
Uygulama, döviz verilerini Frankfurter FREE API üzerinden alır ve LINQ kullanarak çeşitli sorgular yapar.

---

## 🚀 Uygulama Özellikleri

- Canlı döviz verilerini API üzerinden alma  
- Verileri bellekte `List<Currency>` yapısında tutma  
- LINQ ile filtreleme, arama, sıralama ve istatistiksel işlemler  
- Konsol tabanlı menü sistemi  
- `async / await` ve `HttpClient` kullanımı
